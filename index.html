Brython Template

Using Brython, we are able to write and run Python scripts inside the web browser.

How it works:

  • import the brython javascript file
    • <script type="text/javascript"
  src="src/brython.js">
</script>
  • run it when page loads
    • <body onload="brython(1)">
  • add python code
    • <script type="text/python3">
# Python code here
</script>
